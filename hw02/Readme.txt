This folder contains the following files:

etch-a-sketch.c
  This is a simple etch-a-sketch program written in C. It takes input from the keyboard keys w,a,s,d to move the cursor up, left  , down and right respectively, 'x' to exit.
  
push-button-4.js
  This is a javascript program that controls 4 leds via 4 buttons, all connected to the BeagleBone. Each led corresponding to a   button can be turned on/off seperately.
